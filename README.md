# 🤖 Gemini AI Chatbot

A modern, responsive AI chatbot powered by the Google Gemini API, designed to deliver real-time conversational responses with a clean and interactive user interface. This project highlights strong frontend development skills along with seamless API integration.

# 🚀 Overview

This application enables users to interact with an AI model in real time through a smooth and intuitive chat interface. It focuses on performance, usability, and clean design while maintaining a lightweight structure.

The project demonstrates how advanced AI capabilities can be integrated into a responsive web application using modern development practices.

# ✨ Features
💬 Real-time AI-powered conversations
⚡ Smooth typing animation effect
🧠 Chat history persistence (local storage)
🌗 Dark/Light mode toggle
🧹 Clear chat functionality
📱 Fully responsive design
🎯 Clean and interactive user interface
# 🛠️ Tech Stack
`HTML5`
`CSS3`
`JavaScript (Vanilla JS)`
`Google Gemini API`

# 🔑 API Configuration

This project requires a Google Gemini API Key and API URL, which are not included in this repository for security reasons.

To run locally:
Get your API key from Google
Open `script.js`
Add your credentials:
```
const API_KEY = "your_api_key_here";
const API_URL = "your_api_url_here";
```

# ⚠️ Important:

Never share your API key publicly
Do not commit it to GitHub
Use environment variables or a backend for production
